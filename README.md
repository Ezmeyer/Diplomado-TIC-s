# Diplomado-TIC-s
Aplicaciones Prácticas de Arreglos Multidimensionales en Ingeniería

Problema 1: Análisis de Fuerzas en una Estructura 

Este programa utiliza un arreglo bidimensional (matriz 3x3) para representar las fuerzas aplicadas en los nodos de una estructura.
Cada posición de la matriz representa la fuerza aplicada en un nodo específico.
El objetivo del programa es:
Mostrar la matriz de fuerzas.
Calcular la suma total de las fuerzas aplicadas.
Determinar la reacción necesaria en los apoyos para que la estructura esté en equilibrio.
Sumatoria de fuerzas igual a Cero
Por lo tanto, la reacción necesaria será el valor negativo de la suma total de las fuerzas.
Se utiliza un arreglo bidimensional porque permite representar los nodos de la estructura como una tabla o matriz, lo cual facilita:
organizar los datos
recorrerlos con bucles
realizar cálculos matemáticos sobre todos los elementos
Este tipo de estructuras se usa comúnmente en:
análisis estructural
simulaciones físicas
modelado de sistemas de ingeniería


Problema 2: Simulación de Fluido en una Cuadrícula 3D 

Este programa simula el comportamiento de un fluido en un espacio tridimensional utilizando un arreglo tridimensional (3×3×3).
Cada celda del espacio representa un pequeño volumen del fluido y contiene información sobre:
Presión
Temperatura
Velocidad
El objetivo es simular la propagación de una onda de presión a través del fluido, actualizando el valor de presión en cada celda según las presiones de sus celdas vecinas.
Este programa demuestra:
Uso de arreglos 3D
Recorridos con tres bucles anidados
Simulación de fenómenos físicos en un espacio volumétrico
Este tipo de estructuras se utiliza en:
simulaciones de fluidos
videojuegos
modelado físico y científico


Problema 3: Análisis de Imágenes Médicas en 3D 

Este programa simula el procesamiento de imágenes médicas tridimensionales, como las obtenidas en tomografías o resonancias.
Las imágenes se representan como un volumen 3D, donde:
cada capa es una imagen 2D
cada celda es un voxel (píxel en 3D)
El objetivo es aplicar un filtro de promedio para reducir el ruido en el volumen.}
Este programa demuestra:
uso de matrices 3D
procesamiento de datos volumétricos
aplicación de filtros espaciales
Estas técnicas se utilizan en:
tomografías médicas
reconstrucción 3D
procesamiento de imágenes médicas


Problema 4: Manejo de Datos en una Tabla de Sensores 
 
Este programa analiza datos de temperatura provenientes de 5 sensores registrados en 5 momentos diferentes en el tiempo.
Los datos se almacenan en un arreglo bidimensional (5×5) donde:
Las filas representan el tiempo
Las columnas representan los sensores
El objetivo es:
Mostrar los datos en forma de tabla
Calcular el promedio por sensor
Calcular el promedio por momento en el tiempo
Este programa demuestra:
cómo organizar datos en forma de tabla
cómo recorrer filas y columnas con bucles anidados
cómo realizar análisis estadístico básico sobre matrices
Este tipo de procesamiento se usa en:
monitoreo industrial
sistemas IoT
análisis de datos de sensores ambientales

Problema 5: Transformación de Coordenadas en un Sistema Cartesiano 

Este programa trabaja con un conjunto de puntos en un plano cartesiano 2D y aplica una transformación lineal usando una matriz de rotación.
El objetivo es:
representar puntos en un arreglo bidimensional
aplicar una transformación matemática
mostrar los puntos antes y después de la transformación
El programa realiza los siguientes pasos:
Muestra los puntos originales.
Recorre cada punto del arreglo.
Aplica la multiplicación matriz × vector.
Guarda los nuevos puntos en otra lista.
Muestra los puntos transformados.
